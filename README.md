# Architettura e Progetto dei Calcolatori (APC) 💻🏛️

Questo repository contiene il materiale didattico, gli appunti e gli esercizi sviluppati durante il corso di **Architettura e Progetto dei Calcolatori 2026** per la Laurea Magistrale in Ingegneria Informatica.

## 📖 Contenuti
* **Appunti:** Sintesi delle lezioni su gerarchie di memoria, pipeline, set di istruzioni e architetture avanzate.
* **Esercizi:** Implementazioni pratiche, principalmente in **Assembly** e **C**.
* **Simulazioni:** Analisi del comportamento dei processori tramite il **Simulatore ASIM**.

## 🛠️ Tool utilizzati
* **Linguaggi:** Assembly, C.
* **Simulatore:** ASIM.
* **Ambiente:** Windows.
